# Drake Lofi Jeopardy

A lofi-themed, classroom-friendly Jeopardy-style trivia board featuring Drake-related questions. Built with plain HTML/CSS/JS for easy sharing and GitHub Pages deployment.

## How to Run
Open `index.html` in any browser.

## How to Deploy (GitHub Pages)
1. Upload `index.html`, `style.css`, `script.js`, and this `README.md` to your repo.
2. In GitHub: Settings → Pages → Deploy from a branch.
3. Select `main` and `/ (root)`.

## Customize Questions
Edit the `gameData` array in `script.js`.
